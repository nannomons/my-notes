## 🔍 Research Guide for Incoming Stats + Actuarial Science Major

### 🎯 **1. Understand the Landscape**

Start by familiarizing yourself with the fields you're entering:

- **[[Statistics]] Basics**: Learn core ideas—mean, variance, distributions, hypothesis testing.
    
- **[[Actuarial Science]]**: Research what actuaries do (risk assessment, insurance, pensions).
    
- **[[Data Analytics]]**: Understand where it overlaps—actuaries use analytics to make decisions based on data trends.
    

**Start with:**

- [Khan Academy: Statistics and Probability](https://www.khanacademy.org/math/statistics-probability)
    
- SOA/CAS websites for info on exams, tracks, and roles of actuaries

## 🧠 What to Study Before School Starts

### 📊 Core Topics (for Stats + Actuarial)

1. **[[Mathematics]]:**
    
    - [[Calculus (Main)]] (especially integration and series)
        
    - [[Linear Algebra]] (matrix operations, eigenvalues)
        
    - [[Probability Theory]]
        
    - [[Combinatorics]]
        
2. **[[Statistics]]:**
    
    - [[Descriptive Statistics]] vs [[Inferential Statistics]] 
        
    - [[Distributions]]: Normal, Binomial, Poisson, etc.
        
    - [[Estimation and Hypothesis Testing]]
        
3. **Finance (Actuarial-Specific):**
    
    - [[Time Value of Money]]
        
    - [[Annuities and Interest Rates]]
        
    - [[Basic Financial Instruments]] (bonds, stocks)
        

> 📚 Suggested Book: _“Probability and Statistical Inference” by Hogg and Tanis_

## 💻 Should You Learn Data Analytics Through Programming?

**Yes, absolutely.** Data analytics is becoming essential even for actuaries.

### 🔧 Programming Tools to Start With:

| Tool           | Why Learn It First                                                                              |
| -------------- | ----------------------------------------------------------------------------------------------- |
| **[[Python]]** | Versatile, popular in data science, lots of libraries (NumPy, pandas, matplotlib, scikit-learn) |
| **[[R]]**      | Specialized for statistics, still widely used in academia and actuarial work                    |
| **[[SQL]]**    | Essential for data extraction and cleaning from databases                                       |

Start with **Python + pandas** for data manipulation and **matplotlib/seaborn** for visualization.

> 🧪 Try Kaggle's ["Python" and "Intro to Data Analysis"](https://www.kaggle.com/learn) free courses.
