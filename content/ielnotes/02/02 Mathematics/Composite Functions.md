If, say, $(f \circ g)(x)$ is asked of you. The way you're gonna solve it is to solve $f(g(x))$, in other words, input the function $g(x)$ into the function $f(x)$.

Example, given that: 
$f(x) = 3x - 2$
$g(x) = x+4$

$(f \circ g)(x) =$
		$3(x+4)-2$
		$3x+12-2$
		**$3x+10$**

$(f \circ g)(x) = 2x + 10$

---

If $g(x)$ has a given, say, $2$, the equation for $(f \circ g)(x)$ would be $(f(g(2)))(2)$.

Solving it would look like this.

$(f(g(2)))(2)=$
		$g(2) =$
			$2+4$
			$6$
		$f(6)=$
			$3(6)-2$
			$18-2$
			**$16$**
