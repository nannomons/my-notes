## Basic
- [ ] to-do
- [/] incomplete
- [x] done
- [-] canceled
- [>] forwarded
- [<] scheduling

## Extras
- [?] question (?)
- [!] important (!)
- [*] star (*)
- ["] quote (")
- [l] location (l — lowercase L)
- [b] bookmark (b)
- [i] information (i)
- [S] savings (S)
- [I] idea (I — capital i)
- [p] pros (p)
- [c] cons (c)
- [f] fire (f)
- [k] key (k)
- [w] win (w)
- [u] up (u)
- [d] down (d)
- [D] draft pull request
- [P] open pull request
- [M] merged pull request
